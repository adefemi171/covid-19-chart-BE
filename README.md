## How to setup project and run locally

### Clone the repository 

    ```
    git clone https://github.com/adefemi171/covid-19-chart.git
    ```

### Install all dependencies

Using npm

    ```
    npm install
    ```

### To start the App and run locally

    ```
    npm start
    ```
